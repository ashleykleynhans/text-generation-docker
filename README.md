# Docker image for Audiocraft audio processing and generation with deep learning

## Installs

* Ubuntu 22.04 LTS
* CUDA 11.8
* Python 3.10.6
* [Audiocraft](
  https://github.com/facebookresearch/audiocraft)
* Torch 2.0.1
* xformers 0.0.20
